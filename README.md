# ✨ Royal Wedding Invitation Website

A luxurious and fully interactive wedding invitation website designed and developed for the wedding celebration of **Rayan & Simran**.  
This project combines cinematic visuals, elegant UI/UX, responsive design, interactive guest engagement, and modern frontend web technologies to create a premium digital wedding experience.

Guests can generate beautiful AI-style blessings in multiple tones including:

- Sweet & Warm
